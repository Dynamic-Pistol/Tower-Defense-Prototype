# Prototype tower defense game

This is a prototype of an old game of mine, it is made in unity.

The gameplay is simple, double click slots to spawn turrets, double click again to show upgrade menu, then choose an upgrade, you don't consume any coins, and you can move turrets by dragging.

The camera moves by WASD or going near the screen borders.

There is a secret boss, type "supreme" to spawn him.

Nothing is AI-generated, this was made pre-AI and the future product will use 0 AI, only the license which was made recently is AI-generated, and I heavily regret that decision.
